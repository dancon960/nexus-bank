# Reflexión Final: IA y Programación

Tras completar el proyecto TaskFlow (Nexus Bank), estas son mis conclusiones sobre el uso de asistentes inteligentes.

## Puntos Fuertes
La IA me ha ayudado mucho a entender Tailwind CSS sin tener que leerme toda la documentación. También ha sido clave para limpiar el código y que los comentarios sean fáciles de leer.

## Riesgos y Fallos
A veces la IA se inventa clases de CSS que no existen o intenta usar métodos muy modernos que el navegador no reconoce. Si te fías al 100% sin revisar, el proyecto falla.

## Opinión Personal
Para mí, la IA es una herramienta de apoyo, no un sustituto. Prefiero programar con ella para ir rápido en lo visual, pero la lógica del proyecto prefiero gestionarla yo.
