# Ingeniería de Prompts

En esta sección guardo los prompts (instrucciones) que mejor me han funcionado para que la IA me devuelva código sin errores.

## Estrategias Utilizadas
* **Asignación de Rol:** "Actúa como un desarrollador senior experto en Tailwind".
* **Restricciones Claras:** Pedir que no use librerías externas para que el proyecto sea puro JS/CSS.
* **Razonamiento paso a paso:** Pedir que primero explique la lógica y luego escriba el código.

## Prompts Útiles
1. "Refactoriza esta función de LocalStorage para que sea más robusta".
2. "Añade clases de Tailwind para que este botón parezca de una app de Apple".
