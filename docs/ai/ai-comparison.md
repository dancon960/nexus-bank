# Comparativa de programas de IA

En este documento voy a comparar las respuestas y el rendimiento de diferentes IAs (Gemini, ChatGPT y Claude) aplicadas al desarrollo de mi proyecto.

## Conceptos Técnicos Analizados
He pedido a las IAs que expliquen conceptos para entender cómo funciona JavaScript por debajo:
* **El DOM:** Entender cómo JS manipula el HTML.
* **Hoisting:** Por qué algunas variables fallan si no se declaran bien.
* **Event Loop:** Cómo gestiona el navegador los clics y las cargas.

## Pruebas de Debugging
He pasado fragmentos de código con errores típicos de lógica (como no convertir strings a números en el LocalStorage) para ver cuál detectaba el fallo más rápido y daba la solución más clara.

## Conclusión
Comparando las tres, me quedo con Gemini por la claridad de sus ejemplos cotidianos, aunque ChatGPT es muy sólido para funciones largas.
