# Flujo de trabajo con Cursor

Aquí documento cómo he configurado y utilizado el editor Cursor para acelerar el desarrollo de la aplicación.

## Atajos de Teclado Frecuentes
* `Ctrl + K`: Para editar bloques de código con lenguaje natural.
* `Ctrl + L`: Para preguntar dudas sobre el código seleccionado en el chat lateral.
* `Ctrl + I`: Para usar el Composer y hacer cambios en varios archivos a la vez.

## Mejoras Realizadas
1. **Autocompletado de funciones:** Cursor me ayudó a escribir la lógica de los filtros de ingresos/gastos solo con poner un comentario.
2. **Explicación de errores:** Cuando el modo oscuro no funcionaba con Tailwind, el chat contextual me explicó que faltaba configurar el `darkMode: 'class'`.
